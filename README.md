primary feature development still in progress, see below for current status:

* Get User
* Send Pushes
 * Note
 * Link
 * Address
 * Checklist
 * File
   * File Uploads
* Get Devices
* Get Contacts
* Create Contacts
* Update Contact
* Delete Contact
* Subscriptions (add, info, remove)
